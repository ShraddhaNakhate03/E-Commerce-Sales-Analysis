# 🛍️ E-Commerce Sales Analysis 

## 📌 Project Overview
This project focuses on analyzing E-Commerce sales data to extract meaningful insights related to **sales performance, profitability, and customer behavior**.  

The goal is to simulate real-world business scenarios where a Data Analyst identifies trends, evaluates performance, and provides actionable insights for decision-making.

---

## 🎯 Objectives
- Analyze monthly sales performance  
- Identify highest and lowest sales months  
- Evaluate product category and sub-category performance  
- Analyze profit trends across time  
- Understand customer segment behavior  
- Calculate sales-to-profit ratio  

---

## 📊 Dataset Description
The dataset contains transactional sales data with key features such as:

- Order Date  
- Sales Amount  
- Profit  
- Category & Sub-Category  
- Customer Segment  
- Region  

---

## 🔍 Business Problems Solved

### 📅 Monthly Sales Analysis
- Identify highest and lowest sales months

### 📦 Category Analysis
- Find highest and lowest performing categories

### 💰 Profit Analysis
- Determine monthly profit trends  
- Identify most profitable categories and sub-categories  

### 👥 Customer Segment Analysis
- Analyze sales and profit by customer segments  

### 📊 Sales-to-Profit Ratio
- Evaluate efficiency of sales in generating profit
